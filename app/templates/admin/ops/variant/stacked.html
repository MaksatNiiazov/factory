Справочник по шаблону маркировки
Основные
  • Используйте {% verbatim %}{{ имя_переменной }}{% endverbatim %}, чтобы выводить данные переменной. Пример: {% verbatim %}FDM {{ D }}x{{ L }}{% endverbatim %}
  • Вы можете использовать математические операции, + - * / % **. Пример: {% verbatim %}{{ w * h }}{% endverbatim %}
  • Для использование фильтров, используйте символ | после имени переменной. Пример: {% verbatim %}{{ D|upper }}{% endverbatim %}. Список фильтров можно увидеть внизу
Встроенные фильтры
  • abs: Возвращает абсолютное значение
  • float: Превратить значение в число с запятой
  • int: Превратить значение в целое число
  • lower: Возвращает строку в нижнем регистре
  • max: Возвращает максимальное значение из списка. Пример: {% verbatim %}{{ [D, L]|max }}{% endverbatim %}
  • round: Округляет число с плавающей точкой. Можно указать дополнительные параметры: точность и метод. Пример: {% verbatim %}{{ L|round }} или {{ L|round(2, 'floor') }}{% endverbatim %}
    • точность: округлить до той число, что указано.
    • метод: метод округления: common (либо в большую, либо в меньшую сторону), ceil (всегда в большую сторону), floor (всегда в меньшую сторону)
  • trim: Удаляет пробелы слева и справа
  • upper: Возвращает строку в верхнем регистре
Дополнительные фильтры
  • dmy: Форматирует дату в формат ДД.ММ.ГГГГ
  • dmyt: Форматирует дату и время в формат ДД.ММ.ГГГГ ЧЧ:ММ:СС
  • zfill: Добавляет в начале строку нули, пока длина не дойдет до определенного числа. Пример: {% verbatim %}{{ D|zfill(4) }}{% endverbatim %}, если длина D равен 2, то в начале будет добавлен две нули
